SocketSynth

Realtime collaborative synthesizer using websockets.

GitHub

How Does it Work?

On the page, click one of the keys to create a sound. Other people which have the page open will hear that sound. When other people press a key, you will hear it as well.

You can use your keyboard to press keys as well; Use the select below and choose a channel, then use number + letter keys to make sounds. Works best with a QWERTY layout.
